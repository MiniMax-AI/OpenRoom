# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT open a public issue.**
2. Email the maintainers with a detailed description of the vulnerability.
3. Include steps to reproduce if possible.

We will acknowledge your report within 48 hours and provide a detailed response within 5 business days.

## Disclosure Policy

- We will work with you to understand and resolve the issue before any public disclosure.
- We will credit reporters in the fix announcement (unless you prefer to remain anonymous).
